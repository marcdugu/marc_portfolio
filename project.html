🚀 Distinguishing Cosmic String Signals from Noise

🔍 Overview

Detecting gravitational waves from cosmic strings is a challenge due to noise interference. This project utilizes a Convolutional Neural Network (CNN) to classify signals with an impressive 95% accuracy.

🎯 Key Achievements

  • Built a 7-layer CNN to detect cosmic string signals.
  • Trained on simulated Einstein Telescope data.
  • Achieved a 95% classification accuracy.
  • Implemented early stopping & adaptive learning to improve training efficiency.
